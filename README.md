**Телеграм бот тайный санта**

Команды:

start - при deeplinking'е добавляет участника (телеграм пользователя), при обыной команде выдаёт приветствие

startparty - создает пул и делает вас организатором

myprofile - показывает профиль

editprofile - изменяет профиль

draw - начинает жеребьёвку

endparty - удаляет пул
